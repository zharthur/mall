## 项目简介

 本项目是一个基于Vue的仿蘑菇街App的练习项目。

 功能有：主页包括浏览商品、上拉刷新下拉加载更多商品等功能。详情页包括商品的展示，以及商品的参数，评论，店家信息，还有加入购物车功能等。购物车页面是管理添加的商品，是用Vuex保存的数据，并没有提交到数据库。
